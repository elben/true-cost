# True Cost

## Development

(First [install Elm](http://elm-lang.org/install))

```bash
elm-reactor
```

And then open [http://localhost:8000/Main.elm](http://localhost:8000/Main.elm) to see it in action!

### Compiling

```
elm make Main.elm --output truecost.js
open index.html

elm repl
import Main exposing (..)
```
